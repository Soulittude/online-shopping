E-Ticaret oyun satış uygulaması.

Modeller:
  Entity: Veritabanındaki tablo ve sütunlar bu klasörün altındaki sınıflarda belirtiliyor
		-Firma: Firma bilgilerinin bulunduğu entity
		-Platform: Platform bilgilerinin bulunduğu entity
		-Oyun: Oyun bilgilerinin bulunduğu entity
		-Dil: Dil bilgilerinin bulunduğu entity
		-Tag: Tag bilgilerinin bulunduğu entity
		-OyunDil: Oyunların hangi dilleri içerdiğini belirten entity
		-OyunTag: Oyunların hangi tagleri içerdiğini belirten entity
  DTO: Ekleme(POST) işlemleri yapılıyorken, bu klasörün altındaki sınıflar kullanılıyor
		-FirmaDTO: Firma bilgilerinin bulunduğu DTO
		-PlatformDTO: Platform bilgilerinin bulunduğu DTO
		-OyunDTO: Oyun bilgilerinin bulunduğu DTO
		-DilDTO: Dil bilgilerinin bulunduğu DTO
		-TagDTO: Tag bilgilerinin bulunduğu DTO
		-OyunDilDTO: Oyunların hangi dilleri içerdiğini belirten DTO
		-OyunTagDTO: Oyunların hangi tagleri içerdiğini belirten DTO

Controllerlar:
-FirmaController: Firma CRUD işlemleri yapılmasını olanak sağlayan controller
-PlatformController: Platform CRUD işlemleri yapılmasını olanak sağlayan controller
-OyunController: Oyun CRUD işlemleri yapılmasını olanak sağlayan controller
-DilController: Dil CRUD işlemleri yapılmasını olanak sağlayan controller
-TagController: Tag CRUD işlemleri yapılmasını olanak sağlayan controller
-OyunDilController: Oyunların Dil CRUD işlemleri yapılmasını olanak sağlayan controller
-OyunTagController: Oyunların Tag CRUD işlemleri yapılmasını olanak sağlayan controller

Context:
ModelContext: Database bağlantısını kurulduğu yer
